# Release Note

## v0.1.0 - 2026-07-04
### 變更內容
- **2026-07-07 行動端與小工具功能擴展與除錯**：
  - **廠商管理行動端響應式改造**：針對手機畫面（width <= 768px）重構條列式列表（List View）排版，使其自動塊級卡片化、隱藏表頭，並為聯絡人模組加上高質感發光底色與加寬按鈕間距，徹底消除溢出與重疊錯位。
  - **名片掃描器 (OCR) 行動端巨幅升級**：
    - 新增 **「📸 手機相機拍照 (推薦)」** 快捷按鈕，直接啟動原生相機 macro/對焦拍照，完美解決瀏覽器 WebRTC 畫面模糊且不穩的問題。
    - 辨識行新增 **「一鍵快捷分欄 (公司/姓名/電話/信箱/地址)」** 控制按鈕，單手即可瞬間填入資料，不需點選後滾動尋找輸入框。
    - 大號彈窗 (modal-large) 行動端自適應：支援整體頁面依高度比例滾動，固定頁首與頁尾，避免多重滾動條跑版。
  - **知識庫分類目錄行動端點選優化**：將觸控高度擴大至 48px，且在無滑鼠 hover 的手機端強制常開「新增子分類、修改名稱、刪除分類」按鈕，徹底解決原本手機版無法編輯目錄的缺陷。
  - **新增常用小工具大分頁 (Office Tools)**：
    - **批次圖片壓縮工具**：100% 瀏覽器本地端 Canvas 批次多張圖壓縮，支援全域自訂畫質、最大邊長限制比例縮小、一鍵單張或打包下載全部，體積縮減率最高達 90%。
    - **文字排版與清理器**：實作 Windows 換行相容的「贅餘空行清除」、英數「全形轉半形（ＡＢ ⇄ AB）」、「多餘空格清理」、「Markdown 有序/無序列表包裝」以及「簡繁雙向對照翻譯」，徹底修復字體轉換與多行 Windows text 處理的缺陷。
- **2026-07-05 功能擴展與加強**：
  - **工作數據儀表板與熱力圖 (Dashboard)**：新增 SVG 工作熱力圖，模擬 GitHub 熱力圖，支援點擊直接導航至該日期的日誌；整合 Chart.js，支援近 30 天事項狀態（Todo/Success/Failed）比例圓餅圖，與熱門工作標籤前 5 大橫向條形圖。圖表會隨深淺色主題切換而即時重繪並更新配色。
  - **知識庫全文模糊檢索與動態標籤雲**：遍歷 Markdown 全文模糊搜尋，並在搜尋時將卡片標題及內文摘要匹配處進行黃色高亮標記（`<mark>`），自動定位搜尋詞；在分類樹下方實作動態標籤雲，按出現頻率縮放字體，點選可一鍵進行全域標籤過濾。
  - **週報生成器與 Gemini AI 摘要**：新增工作週報生成功能，可聚合選定日期區間的所有工作日誌（分門別類列出 Success/Todo/Failed 及心得），並支援安全儲存個人 Google Gemini API Key；一鍵呼叫 Google 官方 Gemini API 提煉精緻的 Markdown 工作週報，並提供一鍵複製至剪貼簿功能。
  - **廠商 PDF 列印與名片分享拼圖**：編寫專門的 `@media print` 樣式，支援雙欄卡片/表格列印 PDF 輸出並防接縫截斷；整合 HTML5 Canvas 實作「名片分享」功能，自動將廠商資訊及上傳的名片照（正反面）拼合成一張極具質感的數位名片圖供下載分享。
- **專案初始化**：建立基於原生 HTML5/CSS3/ES6 運作的單頁 Web 應用程式 (SPA)。
- **本地資料庫**：採用 IndexedDB 儲存筆記、廠商與知識庫資料，確保資料本機隱私。
- **Google Drive 同步**：整合 Google Identity Services 進行 OAuth 認證，支援 appDataFolder 本地/雲端自動 Debounced 同步與手動備份。
- **全域雙主題切換**：
  - 新增側邊欄左下角「切換主題」按鈕，支援在「深色炫酷模式」與「柔和淺色模式」間無痛切換。
  - 淺色模式特調為護眼、對眼睛柔和的「燕麥紙張色 (Oatmeal/Warm Gray)」，文字為深石墨褐以降低對比刺眼感。
  - 支援 localStorage 記住您的主題選擇，重新整理或重啟瀏覽器自動保持您的主題設定。
- **每日工作日誌與進度追蹤系統 (原智能筆記本)**：
  - 將隨手記重構為以「天」為單位的日誌卡片，同日工作自動聚合展示。
  - 新增今日日誌編輯器，支援事項「進行中」、「已完成」與「⚠️ 未成功/遇阻」狀態管理。
  - 支援卡關/未成功原因登錄，並於歷史日誌中以醒目紅色區塊警示。
  - 頂部加入工作狀態統計看板（本日完成率、進行中工作數、遇阻事項數），支援點擊進行中/遇阻一鍵過濾日誌。
  - 支援歷史卡片直接勾選切換狀態、Enter 快捷鍵新增以及存檔防呆自動補登。
  - 新增**日期選擇器 (Date Picker) 支援任意日期補登**：取代靜態的「今日」，允許切換日期以加載、補登或修改歷史工作日誌；點擊歷史卡片日期可直接載入編輯。
  - 新增**工作日誌 Excel 報告匯出功能**：整合 ExcelJS，支援依據狀態（進行中/已完成/遇阻）進行色彩標註（如卡關紅底、已完成刪除線），並自動產出帶有彙整完成率的日誌報告。
  - 內建 100% 舊筆記向下相容轉化機制，無痛移轉既有資料。
- **廠商管理系統**：
  - 導入 **Tesseract.js OCR**，支援手機拍照與手動檔案上傳。
  - 整合 **OTSU（大津演算法）二值化影像預處理**，提升中英文辨識準確率。
  - 新增 **OCR 閥值微調滑桿**，允許用戶即時調整黑白對比並重新辨識。
  - 修正 CSS 排版，避免掃描器滑桿及控制按鈕遭溢出裁切。
  - 修正名片照片預覽置中縮放，防止高解析圖片超出容器。
  - 升級為**多相片管理系統**，支援多張名片照片（含正反面與報價單）上傳、縮圖網格切換與單張刪除。
  - 新增**多聯絡人與職稱管理**：支援同一家廠商記錄多名聯絡人（含姓名、職稱、電話、Email），並可自由切換主聯絡人，透過後台架構完美向下相容舊有資料庫欄位。
  - 新增**代理原廠品牌 (Brands) 管理**：可管理多個代理品牌並顯示為精美微晶片標籤。
  - 新增**品牌與多聯絡人全域搜尋**：打通主搜尋欄，輸入「Ti」或聯絡人職稱均可自動模糊檢索出關聯廠商。
  - 新增**卡片與條列式表格 (Table) 雙重視角切換**：新增工整對齊的條列式表格視角，支援自適應與響應式排版，使大螢幕資訊呈現更為清晰。
  - 新增**Excel 報表樣式美化匯出**：整合 ExcelJS，依據全域常備技能 [excel_export_stylizer](file:///C:/Users/Andy/.gemini/config/skills/excel_export_stylizer/SKILL.md) 匯出當前過濾狀態下之廠商報表，具備凍結、自動適應中文字寬、靛藍漸層表頭、偶數列斑馬紋與等寬字體。
  - 新增**拜訪歷史紀錄 (Visit Logs) 管理**：
    - 在廠商編輯視窗中整合拜訪紀錄面板，支援選擇來訪聯絡人（下拉選單自動連動已錄入的 Contacts 名單）與輸入來訪事由，並以垂直時間軸 (Timeline) 進行渲染。
    - 主畫面卡片與表格上新增來訪次數小徽章（Visit Badge），點選後免進入編輯，即可直接彈出快速檢視浮窗，閱讀完整的歷史來訪日程與事由時間軸。
    - 打通搜尋與報表匯出：搜尋來訪事由或來訪人可精準過濾廠商，且匯出 Excel 時會自動包含來訪總次數與最近一次拜訪明細。
- **客製化彈窗系統**：全面替換瀏覽器原生 `alert()` 與 `confirm()`，採用 Promise-based 非同步全域對話框，具備高質感毛玻璃與彈性動態縮放效果，維持全站視覺一致性。
- **全域圖片放大檢視對話框 (Custom Lightbox)**：
  - 支援點擊名片大圖、列表名片按鈕直接展開全螢幕 Lightbox 預覽。
  - 實作滑鼠滾輪 **0.5x ~ 6x** 無極縮放，並於放大狀態下支援滑鼠拖曳平移。
  - 支援多張照片之左/右按鈕切換與鍵盤左右箭頭 (`ArrowLeft`/`ArrowRight`) 切換，且內建底部張數指示器。
  - 封裝為全域 Gemini 技能常備檔 [image_lightbox_viewer](file:///C:/Users/Andy/.gemini/config/skills/image_lightbox_viewer/SKILL.md)，作為後續其他專案之重用資產。
- **知識庫分類管理**：
  - 支援兩層分類目錄樹狀結構，整合 Marked.js 渲染 Markdown 文章。
  - 新增**分類目錄折疊收合**：在有子類的大類左側新增旋轉折疊 chevron 箭頭，點選可折疊或展開子項目，狀態會自動透過 localStorage 持久化保存。
  - 新增**知識庫模糊搜尋**：選取分類後，在右側新增搜尋輸入框，支援即時依標題、內文、標籤搜尋篩選卡片，無符合結果時提供貼心提示。
  - 升級**Markdown 編輯器與富文本輔助工具列**：
    - 新增編輯器頂部「編輯內容」與「即時預覽」雙頁籤切換，方便對 Markdown 不熟悉的用戶即時確認渲染效果。
    - 內建 Markdown 快捷格式化工具列，支援一鍵加粗、斜體、下劃線、無序列表、程式碼區塊、大標題與中標題。
    - 支援文字編輯功能：新增**文字顏色**（紅、藍、綠、橘、紫）與**字體大小**（超大 24px、大 20px、小 14px）下拉套用選單，自動插入 HTML span 樣式，使字體美化輕鬆直覺。
- **自動化測試**：內建 `js/tests.js` 整合與單元測試，支援 `?test=true` 條件執行。
