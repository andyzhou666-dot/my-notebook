設定與同步

Google Drive 雲端同步

將您的所有筆記、廠商與知識庫資料自動備份至個人 Google Drive 專屬空間，讓公司、家裡與手機保持資料同步。

未連線 請登入 Google 帳號啟用同步

* 由於預設 Client ID 屬於測試開發帳戶，跨網域登入時會遇到 401 錯誤。請前往 Google Cloud Console 申請您的專屬 OAuth Client ID（需授權 JavaScript 來源為您的當前網址/網域，如 http://localhost:5000 或 GitHub 網址），貼上後儲存再點擊登入。

Gemini AI 助理設定

設定您的 Google Gemini API 金鑰後，系統可以在產生週報時，直接呼叫 AI 幫您自動整理與提煉工作心得摘要。

* API Key 將安全儲存於您本地瀏覽器的 LocalStorage 中，摘要請求直接從您的瀏覽器發送至 Google 官方 API 節點，絕不會經過任何第三方中轉。

手動本地備份與還原

您也可以隨時將完整的資料庫匯出成單一的 JSON 檔案至本地保存，或是進行手動資料移轉。
